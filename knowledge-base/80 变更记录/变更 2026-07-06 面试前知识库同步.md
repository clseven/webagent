# 变更 2026-07-06 面试前知识库同步

## 背景

本次同步发生在代码长期快速演进之后，原知识库仍停留在 2026-06 中旬 MCP 初版和早期文件工作区阶段，已经无法支撑简历和面试准备。

## 本次覆盖的代码事实

- Agent 编排已拆分为上下文准备、规划服务、ReactAgent 工厂和 Hook 装配服务。
- ReactAgent 支持 PreToolUse、PostToolUse、Stop Hook。
- `todo_write`、`AgentTodoService` 和 `FinalTodoGuardHook` 形成运行时 TodoState 闭环。
- `run_subagent` 可 fork 子 Agent，并支持后台任务结果收集。
- `view_image` 通过 PostToolUseHook 调用 Agnes 视觉模型观察图片。
- MCP Client 默认启用，支持系统级 stdio/streamable-http 和用户动态 shell transport。
- Sandbox endpoint 管理正在从会话级收敛为用户级。
- 新增 `/api/sessions/{id}/aio/view-url` 和 `/sandbox-view/{token}/...` 同源视图代理。
- 前端 Chat 页面升级为带右侧工具 dock 的工作台，支持沙箱视图、工作区、图片图库和文件预览。
- RAG Milvus 默认关闭，Noop 向量服务支持无 Milvus 开发启动。
