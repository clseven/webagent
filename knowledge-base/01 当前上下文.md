## 当前系统事实

- 后端：Spring Boot 3.2.5、Java 17、Maven、Spring Web、WebFlux、WebSocket、Spring Data JPA。
- 前端：`src/main/resources/static` 下的静态 Vue 3 + Vue Router，无独立 Node 构建步骤。
- 持久化：MySQL + JPA；上传和知识库原文默认落在 `./uploads`。
- Agent 主模型：当前 `deepseek` profile 下 planner/executor 都使用 DeepSeek `deepseek-v4-pro`；`plannerLlm` 仍保留给标题等轻量调用。
- 视觉模型：`visionLlm` 使用 Agnes `agnes-2.0-flash`，只负责 `view_image` 后的图片观察，不直接接管最终回复。
- RAG：Embedding 使用智谱 `embedding-3`，维度 2048；Milvus 默认 `RAG_MILVUS_ENABLED=false`，启用后集合为 `knowledge_chunks_v3`。
- Query Rewrite：默认回退主流程 DeepSeek 配置；Rerank 默认独立使用 DeepSeek `deepseek-v4-flash`，可通过 `RAG_RERANK_PROVIDER=bge` 切回本地 BGE 服务。
- MCP：`agent.mcp.enabled` 默认 true；支持系统级 stdio/streamable-http，用户动态配置还支持 shell transport。
- 子代理：`agent.sub-agent.enabled` 默认 true，类型包括 analyzer、searcher、browser、general，运行时由 `run_subagent` 工具触发。
- 沙箱：当前以用户级 AIO Sandbox 为中心，多个会话共享同一用户沙箱；浏览器视图通过同源代理暴露。
