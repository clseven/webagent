# 前端模块

## 1. 形态

前端是 Spring Boot 静态资源目录中的 Vue 3 单页应用：

- 无 Node 构建步骤，无独立 `package.json`。
- 使用全局变量形式注册组件。
- 使用 Hash Router。
- API 与后端同源部署，`API_BASE` 为空。
- `index.html` 通过版本号 query 控制静态资源缓存。
- `fetch + ReadableStream` 处理 SSE（需要携带 Authorization Header）。
- Highlight.js CDN 用于代码高亮。

## 2. 页面

| 路由 | 页面 | 主要能力 |
| --- | --- | --- |
| `/login` | Login | 注册和登录 |
| `/chat` | Chat | 应用选择、会话、上传、SSE 对话、工具行动流、沙箱视图、工作区、文件产物 |
| `/apps` | AgentApps | 配置 Agent 应用、知识库和 Skill |
| `/skills` | Skills | 设置 Skill 根目录、查看和启用 Skill |
| `/knowledge` | Knowledge | 知识库、文档、搜索和预览 |
| `/mcp` | Mcp | MCP 管理入口 |
| `/token-stats` | TokenStats | Token 汇总和图表 |

## 3. Chat 页面布局

Chat 页面是工作台式布局：

- 聊天主区域展示历史、实时思考、工具调用、观察结果和最终回答。
- 左侧是会话列表和应用选择。
- 右侧工具 dock 可打开"沙箱"和"工作目录"。
- 沙箱视图支持浏览器、终端、VSCode 和文件四个入口。
- 工作目录复用 `WorkspaceBrowser` 嵌入 dock。
- dock 宽度可拖拽调整；拖拽时临时禁用 iframe 鼠标事件，避免沙箱页面抢走拖拽。
- 移动端宽度较小时隐藏 dock。

`MainLayout` 在 `/chat` 路由进入 `chat-shell-mode`，隐藏全局顶栏和旧的全局 `WorkspaceBrowser` 浮动入口，让 Chat 页面自己拥有完整工作台布局。

## 4. SSE 和事件展示

`api.createChatStream` 使用 `fetch + ReadableStream`（原生 `EventSource` 不方便携带 Authorization Header）：

- 按空行拆分 SSE frame。
- 合并连续 token，按动画帧刷新。
- 保证阶段事件（plan、tool_call）不被大量 token 延后。
- 支持用户停止流式请求。
- 使用 `AgentActionNarrator` 的工具行动说明展示"正在查看图片""正在更新任务进度"等可读状态。

事件类型：`plan`、`thinking`、`tool_call`、`observation`、`answer`、`status`、`interrupted`、`error`、`done`。

## 5. 文件与产物体验

- 上传文件以卡片方式展示。
- assistant 产生的正式文件产物进入 `chat-artifacts` 区。
- 图片产物最多先展示 4 张，多余图片折叠为数量提示。
- `browser_screenshot` 只有 `deliver_to_user=true` 时展示。
- `download_file` 是正式交付文件；存在正式文件时会抑制正文里重复的旧式文件链接卡片。
- `FilePreviewer` 统一处理文本、代码、Markdown、CSV、图片、PDF、Office 转 PDF 和知识库切片。

## 6. 会话切换

会话切换会重置沙箱视图 URL，并在需要时重新获取 view-url。沙箱视图 token 只绑定 userId，切换会话不会导致沙箱重建。

## 7. 风险与排查

| 现象 | 优先检查 | 说明 |
| --- | --- | --- |
| SSE 连接断开 | token 是否过期、网络 | fetch + ReadableStream，断开后不可自动重连 |
| 沙箱视图 iframe 空白 | view-url 是否过期、token 是否有效 | token 30 分钟过期，需要重新获取 |
| dock 拖拽卡顿 | iframe 鼠标事件是否临时禁用 | 拖拽时需要禁用 iframe 事件 |
| 图片不展示 | `deliver_to_user` 是否为 true | 只有 true 的截图进入卡片组 |
| 静态资源缓存 | index.html 版本号 query | 版本号变化时浏览器重新加载 |
| 移动端 dock 消失 | 屏幕宽度 | 移动端宽度较小时隐藏 dock |
