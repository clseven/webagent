# 工具目录

## Shell 与生命周期

| 工具 | 实现 | 作用 |
|---|---|---|
| `execute_command` | `ExecuteCommandTool` | 在 AIO Sandbox 中执行 shell 命令 |
| `shell_wait` | `ShellWaitTool` | 等待异步 shell 任务 |
| `shell_kill` | `ShellKillTool` | 终止 shell 任务 |
| `request_sandbox` | `RequestSandboxTool` | 请求或确认沙箱资源 |

## 文件与文档

| 工具 | 实现 | 作用 |
|---|---|---|
| `list_files` | `ListFilesTool` | 列出目录文件 |
| `read_file` | `ReadFileTool` | 读取文件内容 |
| `write_file` | `WriteFileTool` | 写入文件 |
| `file_search` | `FileSearchTool` | 搜索文件 |
| `file_replace` | `FileReplaceTool` | 简单替换文件内容 |

## 浏览器与视觉

## MCP 和运行时控制

| 工具 | 实现 | 作用 |
|---|---|---|
| `mcp_list_servers` | `McpListServersTool` | 查看 MCP server 配置和状态 |
| `mcp_add_or_update_server` | `McpAddOrUpdateServerTool` | 新增或更新用户 MCP server |
| `mcp_remove_server` | `McpRemoveServerTool` | 移除用户 MCP server |
| `mcp_reload` | `McpReloadTool` | 重新加载 MCP 配置 |
| 动态 MCP 工具 | `RealMcpTool` | 由 MCP Server 暴露，命名按 server/tool 适配 |
| `run_subagent` | `RunSubagentTool` | fork 子 Agent 执行子任务 |
| `todo_write` | `TodoWriteTool` | 更新会话内 TodoState |

## 注意事项

- 当前 AIO 路径下主要使用 AIO/ALL 类工具；COMMON 工具需要确认过滤策略后再依赖。
- `browser_screenshot` 默认不展示到最终图片卡片组，除非参数显式设置 `deliver_to_user=true`。
- MCP 新增 server 后，本轮不一定立即出现新工具；通常下一条用户消息或下一轮 Agent loop 后才可用。
- `run_subagent` 禁止递归创建子代理，并按子代理类型限制工具集合。
