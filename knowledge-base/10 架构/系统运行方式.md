# 系统运行方式

## 1. 快速结论

- 项目以单体 Spring Boot 方式运行，默认端口 `8081`。
- 默认激活 profile 是 `deepseek,local`，规划和执行默认走 DeepSeek，视觉观察默认走 Agnes。
- 前端静态文件、后端 API、SSE、沙箱视图代理都由同一个应用提供。
- 每个用户复用一个长期 AIO Sandbox，多个会话共享该 Sandbox。
- Milvus 默认关闭，RAG 文档仍可上传、解析、切片，但向量检索会降级为空结果。

## 2. 启动形态

主配置：

```yaml
server:
  port: 8081

spring:
  profiles:
    active: deepseek,local
```

静态前端位于 `src/main/resources/static`，默认与后端同源访问。开发环境可通过 `application-dev.yml` 把静态资源指向源码目录，便于前端改动即时生效。

## 3. 运行依赖

| 依赖 | 当前用途 | 默认状态 |
| --- | --- | --- |
| MySQL | 用户、会话、应用、消息、知识库、文档、用户沙箱映射 | 必需 |
| AIO/OpenSandbox | 用户级沙箱、shell、文件、浏览器、noVNC、code-server | 必需 |
| DeepSeek | 默认 planner/executor LLM | 需要 API Key |
| Agnes | 默认 vision LLM | 需要 API Key |
| 智谱 Embedding | RAG 文档和查询向量化 | 需要 API Key |
| Milvus | 向量召回 | 默认关闭 |
| DeepSeek Rerank | RAG 默认 rerank provider | 使用独立 `RAG_RERANK_MODEL=deepseek-v4-flash` |
| BGE Rerank | RAG 可选 rerank provider | 仅 `RAG_RERANK_PROVIDER=bge` 时使用，默认地址 `http://127.0.0.1:8002/rerank` |

## 4. 对话运行主线

1. 前端调用 `POST /api/sessions` 创建会话。
2. 后端异步创建或复用用户级 Sandbox。
3. 用户发送消息，前端优先走 SSE 流式接口。
4. `AgentTurnContextService` 准备历史、上传文件上下文、应用配置、Skill prompt、知识库增强和工具上下文。
5. `AgentPlannerService` 生成计划。
6. `ReactAgentFactory` 创建执行器。
7. `ReactAgentHookService` 注册图片观察、最终回答门禁、子代理等 Hook。
8. `ReactAgent` 执行 ReAct 工具循环，流式返回 plan、thinking、tool_call、observation、answer 等事件。
9. 完成后保存 assistant 消息、事件列表和 token 用量。

## 5. 沙箱运行主线

1. 应用启动时，`SandboxServiceImpl` 从 `user_sandbox` 恢复用户与 Sandbox 绑定。
2. 通过 AIO 健康探针确认 endpoint 是否仍可用。
3. 新会话优先复用健康的用户 Sandbox。
4. 不健康时清理映射并重建 Sandbox。
5. AIO endpoint 注册到用户级 key `__user_{userId}`，避免同一用户多会话缓存出不同 endpoint。
6. 浏览器视图通过 `GET /api/sessions/{id}/aio/view-url` 签发短期 token。
7. 前端 iframe 访问 `/sandbox-view/{token}/...`，由后端同源代理 HTTP 和 WebSocket。

## 6. 关键配置

| 配置项 | 当前默认值 | 说明 |
| --- | --- | --- |
| `server.port` | `8081` | 应用端口 |
| `spring.servlet.multipart.max-file-size` | `100MB` | 单文件上传上限 |
| `agent.sandbox.domain` | `localhost:8080` | OpenSandbox 服务地址 |
| `agent.sandbox.image` | `agent-infra/sandbox-office:latest` | 默认 AIO 镜像 |
| `agent.sandbox.sandbox-timeout` | `P1D` | 沙箱存活时间 |
| `agent.sub-agent.enabled` | `true` | 是否启用 `run_subagent` |
| `rag.milvus.enabled` | `false` | 是否启用 Milvus |

## 7. 常见排查

| 现象 | 优先检查 |
| --- | --- |
| 服务启动失败 | MySQL 连接、profile、API Key 配置 |
| 创建会话慢 | AIO Sandbox 创建、镜像拉取、ready timeout |
| SSE 无响应 | `AgentServiceImpl.chatStream`、LLM API、工具阻塞 |
| 沙箱视图 502 | token 是否过期、用户 endpoint 是否健康、代理 Location 改写 |
| RAG 检索为空 | `rag.milvus.enabled`、Embedding Key、Milvus 集合 |
