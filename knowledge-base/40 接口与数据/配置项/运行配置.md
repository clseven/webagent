# 运行配置

## 服务基础配置

- `app.base-url`：默认 `http://localhost:8081`。
- `server.port`：默认 `8081`。
- `spring.profiles.active`：当前默认 `deepseek,local`。
- Multipart：单文件和请求最大 100MB。

## 数据库

- `MYSQL_URL`：默认 `jdbc:mysql://localhost:3306/sandbox_agent?...`。
- `MYSQL_USERNAME`：默认 `root`。
- `MYSQL_PASSWORD`：默认空。
- JPA `ddl-auto=update`。

## Agent MCP

- `MCP_ENABLED`：默认 true。
- `MCP_REQUEST_TIMEOUT`：默认 60s。
- `MCP_INIT_TIMEOUT`：默认 30s。
- `MCP_MAX_USER_SERVERS`：默认 10。
- `MCP_USER_ALLOW_HTTP`：默认 false。
- `MCP_USER_ALLOW_PRIVATE_NETWORK`：默认 false。
- 系统级 servers 支持 stdio 和 streamable-http；用户动态配置额外支持 shell。

## 搜索

- `SEARCH_MAX_RESULTS`：默认 5。
- `SEARCH_TIMEOUT`：默认 10s。
- `SEARCH_PROXY_HOST` / `SEARCH_PROXY_PORT`：默认 `127.0.0.1:7890`。
- Baidu Search 可通过 `BAIDU_SEARCH_API_KEY` 等变量配置。

## 子代理和后台任务

- `SUB_AGENT_ENABLED`：当前默认 true。
- `SUB_AGENT_TIMEOUT`：默认 120 秒。
- 子代理类型：analyzer、searcher、browser、general。
- `BG_MAX_CONCURRENT`：后台任务线程池默认 5。

## Sandbox

- `SANDBOX_DOMAIN`：默认 `localhost:8080`。
- `SANDBOX_IMAGE`：默认 `agent-infra/sandbox-office:latest`。
- `agent.sandbox.ready-timeout`：默认 PT180S。
- `SANDBOX_TIMEOUT`：默认 P1D。

## 存储和 Skill

- `STORAGE_TYPE`：默认 local。
- `STORAGE_LOCAL_PATH`：默认 `./uploads`。
- `SKILL_DIR`：默认 `.claude/skills`。
- OSS 配置存在但当前本地存储是主路径。

## LLM

默认 `application.yml`：

- planner：DeepSeek `deepseek-v4-flash`。
- executor：DeepSeek `deepseek-v4-flash`，`DEEPSEEK_THINKING_ENABLED=false`。
- vision：Agnes `agnes-2.0-flash`。

Profile：

- `application-deepseek.yml`：planner/executor 使用 DeepSeek。
- `application-agnes.yml`：planner/executor 使用 Agnes；vision 仍由 `application.yml` 控制。

## RAG

- `RAG_PREVIEW_CONVERSION_ENABLED`：默认 true。
- `RAG_PREVIEW_CONVERSION_TIMEOUT_SECONDS`：默认 120。
- `RAG_EMBEDDING_MODEL`：默认 `embedding-3`，维度 2048。
- `RAG_MILVUS_ENABLED`：默认 false。
- `MILVUS_HOST` / `MILVUS_PORT`：默认 `localhost:19530`。
- `RAG_REWRITE_MODEL`：默认 `qwen3:1.7b`。
- `RAG_RERANK_URL`：默认 `http://127.0.0.1:8002/rerank`。
